Welcome to Survey & GraphDB Manager

A comprehensive solution for exporting survey data from LimeSurvey, storing it in GraphDB knowledge graphs, and building new surveys from existing data.

LimeSurvey Export

Connect to your LimeSurvey instance and export surveys, questions, groups, and responses in multiple formats (JSON, CSV).

  • ✅ List all surveys
  • ✅ Export questions and groups
  • ✅ Export participant responses
  • ✅ Get survey properties and statistics

GraphDB Input

Upload ontologies and survey data to GraphDB repositories, creating structured knowledge graphs for advanced querying.

  • ✅ Create and manage repositories
  • ✅ Upload OWL ontologies
  • ✅ Import CSV data with RML mapping
  • ✅ View loaded named graphs
Open GraphDB Manager →
🔨

Survey Builder

Query your GraphDB knowledge graph, select questions and groups, and create new surveys directly in LimeSurvey.

  • ✅ Query GraphDB with SPARQL
  • ✅ Browse groups and questions
  • ✅ Drag-and-drop survey composition
  • ✅ Create surveys in LimeSurvey
Open Survey Builder →

📋 Typical Workflow

1
Export from LimeSurvey

Use the LimeSurvey page to connect and export your survey structure (groups, questions, properties).

2
Upload to GraphDB

Use the GraphDB page to create a repository, upload your ontology, and import the CSV data.

3
Query and Compose

Use the Survey Builder to query your knowledge graph, select questions, and organize them.

4
Create New Survey

Export your composition or create a new survey directly in LimeSurvey.

Choose where you want to start. Each module is independent, so you can start from any point in the workflow.
